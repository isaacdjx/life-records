# 行程规划助手

一个轻量级的旅行行程规划工具，输入目的地、天数、预算和同行人员，一键生成详细的每日行程安排。

## 功能特性

- 表单输入：目的地、天数（1-30天）、预算（经济/中档/高端）、同行人员
- 智能行程生成：内置北京、上海、成都、西安、厦门 5 个城市景点模板
- 每日时段：上午、中午、下午、晚上，每个时段推荐景点和活动
- 费用估算：根据预算档位自动调整花费
- 高德导航：一键跳转高德地图导航到景点
- 天数切换：滑动切换查看每天的行程
- 重新生成：不满意可一键重新生成

## 部署

单文件静态站，直接部署 `index.html` 到任意静态托管服务：

- GitHub Pages
- Vercel
- Netlify
- 或直接双击打开

## 项目结构

```
records/
├── index.html    # 主页面（含所有样式和逻辑）
└── README.md     # 本文件
```

## 使用方式

1. 打开 `index.html`
2. 填写目的地（默认北京）
3. 设置天数（默认 5 天）
4. 选择预算和同行人员
5. 点击"生成行程"
6. 切换天数 tab 查看每天安排
7. 点击"去高德导航"跳转导航
